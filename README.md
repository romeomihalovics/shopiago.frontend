# shopiago.frontend

Sample Frontend Project with Vue
![Preview](https://media.giphy.com/media/KBVJN3QWekmpj2i6Gq/giphy.gif)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
